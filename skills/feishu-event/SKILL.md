# event (v1)

## 概述

飞书事件订阅通过 WebSocket 长连接实时监听飞书平台事件（消息、通讯录变更、日历变更等）。

在 openclaw-lark 插件中，事件接收由插件的 channel 层自动处理，无需手动配置 WebSocket 连接。

## 事件类型

飞书支持的事件类型包括：
- **消息事件**：新消息、消息已读、消息撤回
- **通讯录事件**：用户入职/离职、部门变更
- **日历事件**：日程创建/更新/删除
- **审批事件**：审批状态变更
- **视频会议事件**：会议开始/结束
- **云文档事件**：文档创建/更新/权限变更

## 配置说明

事件订阅需要在飞书开放平台的应用管理后台进行配置：
1. 进入应用的「事件订阅」页面
2. 选择需要订阅的事件类型
3. 配置事件接收方式（长连接模式由插件自动处理）

## 注意事项

- 事件订阅需要对应的权限 scope，请确保应用已申请相关权限
- 事件数据可能包含敏感信息，处理时需遵循安全规则
- 插件会自动重连断开的 WebSocket 连接
